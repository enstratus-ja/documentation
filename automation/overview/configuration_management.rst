構成管理の追加
--------------

..
    An optional configuration management execution call configured for each launch
    configuration if Chef/Puppet is used to configure servers.

サーバーを設定するのに Chef/Puppet を使うなら、それぞれの起動構成に任意の構成管理呼び出しを設定します。
